# Day 2 – Data Visualization and Storytelling

## Objective

To create meaningful visualizations that convey business insights using Power BI.

## Dataset

SuperStoreOrders.csv

Dashboard Highlights

Total Sales: $13M

Total Profit: $1.47M

Total Quantity Sold: 178K

## Key Insights

Technology is the highest revenue-generating category.

Some sub-categories operate at a loss.

Sales show consistent growth over time.

The top 5 regions contribute the majority of total sales.

## Tools Used

Power BI Desktop

The dashboard focuses on clean design, proper visual hierarchy, and business storytelling.
